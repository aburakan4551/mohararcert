## 🌐 نظرة عامة على المشروع

يهدف هذا المشروع إلى حل مشكلة الجهد والوقت المستغرقين في كتابة وتصميم وتوزيع شهادات الشكر والتقدير يدوياً. يوفر النظام محررًا بصريًا متكاملاً يمكّن المستخدم من رفع قالب الشهادة (صورة أو PDF)، وتحديد مواقع الحقول النصية (مثل الاسم، اللقب، التاريخ، المناسبة)، مع إمكانية استيراد آلاف الأسماء من ملف Excel واحد وتوليد الشهادات مطبوعة بضغطة زر واحدة.

## ⚡ المميزات البارزة

1. **🖥️ محرر بصري تفاعلي (WYSIWYG Editor):**
   * سحب وإفلات العناصر والنصوص في أي مكان على لوحة التصميم.
   * التحكم الكامل بالخطوط (الحجم، اللون، نوع الخط، المحاذاة، السماكة).
   * إضافة وإدارة الطبقات (نصوص، أختام، تواقيع، رموز استجابة سريعة).

2. **📊 استيراد جماعي عبر Excel (Bulk Processing):**
   * إدخال مئات أو آلاف المستفيدين دفعة واحدة عبر رفع ملف Excel.
   * مطابقة أعمدة Excel تلقائياً مع حقول التصميم المحددة.
   * توليد فوري لجميع الشهادات بضغطة زر.

3. **🔗 التحقق الرقمي والـ QR Code:**
   * إنشاء رمز استجابة سريعة (QR Code) تلقائي لكل مستفيد.
   * إمكانية ربط رمز الـ QR برابط سجل الشهادة أو صفحة التحقق الخاصة بالجهة المصدرة.

4. **📜 دعم قوالب الـ PDF والصور:**
   * رفع القوالب كصور أو ملفات PDF مباشرة، ويقوم النظام بتحويلها وعرضها كخلفية للوحة التحرير بدقة مذهلة.

5. **🛡️ حماية وبوابة أمان:**
   * تأمين لوحة التحكم والعمليات الحساسة ببوابة تسجيل دخول وحماية مدمجة.

6. **🔢 نظام الأرقام التسلسلية الذكي:**
   * توليد أرقام تسلسلية متسلسلة للشهادات مع إمكانية تخصيص البادئة (Prefix) مثل (`CERT-2026-0001`).

7. **📋 سجل مركزي للشهادات (Registry):**
   * تخزين محلي لجميع الشهادات الصادرة لتسهيل البحث عنها، طباعتها مجدداً، أو تصدير بياناتها.

8. **🖨️ طباعة وتصدير عالي الدقة:**
   * تصدير الشهادات كملفات PDF عالية الدقة مناسبة للطباعة الفعلية دون خسارة الجودة.

## 📈 سيناريوهات الاستخدام العملي

### 1️⃣ إصدار شهادة فردية
1. انتقل إلى صفحة **إنشاء شهادة**.
2. اختر القالب الأساسي المناسب أو ارفع قالبك الخاص كصورة أو PDF.
3. قم بإضافة الحقول النصية (الاسم، النص، التاريخ) واسحبها للموقع المطلوب.
4. خصص الخطوط والألوان وأضف التوقيع والختم الرقمي.
5. انقر على **تصدير كـ PDF** أو **طباعة فورية**.

### 2️⃣ التوليد الجماعي (دفعة واحدة عبر Excel)
1. قم بإعداد ملف Excel يحتوي على قائمة بالأسماء والبيانات المطلوبة (على سبيل المثال: عمود `الاسم` وعمود `اللقب`).
2. اذهب إلى صفحة **استيراد Excel**.
3. ارفع ملف الـ Excel وسيقوم النظام بقراءة الأعمدة تلقائياً.
4. حدد حقول الربط المتغيرة بين لوحة التصميم وأعمدة الـ Excel.
5. عاين الناتج، ثم انقر على **توليد الكل** ليقوم النظام بإخراج حزمة الشهادات كاملة دفعة واحدة.

## 🛠️ تفاصيل الإعداد والتخصيص

* **الملفات المحلية (.env):** المشروع يعمل بالكامل على جانب العميل (Client-Side) ولا يتطلب قاعدة بيانات خارجية بشكل إجباري، مما يعني سهولة استضافته على سيرفرات الاستضافة الساكنة.
* **البيانات والذاكرة المحلية:** يتم حفظ الشهادات الصادرة وسجل الأرشيف وحالة الإعدادات تلقائياً في ذاكرة المتصفح المحلية (`localStorage`) للحفاظ على البيانات حتى بعد إغلاق المتصفح.

## 🏗️ النشر والتشغيل في بيئة الإنتاج (Production)

يمكن بناء المشروع وتصديره كملفات Static جاهزة للنشر على أي منصة استضافة مثل (Vercel, Netlify, GitHub Pages, Nginx):

1. **إنشاء نسخة الإنتاج المجمّعة:**
   ```bash
   npm run build
   ```
   *سيتولد مجلد باسم `dist` يحتوي على ملفات HTML, CSS, JS مجمعة ومحسنة كلياً.*

2. **النشر السريع باستخدام Vercel:**
   ```bash
   npm install -g vercel
   vercel --prod
   ```

3. **التشغيل باستخدام Nginx (ملف إعداد بسيط):**
   ```nginx
   server {
       listen 80;
       server_name yourdomain.com;
       root /var/www/mohararcert/dist;
       index index.html;

       location / {
           try_files $uri $uri/ /index.html;
       }
   }
   ```

## 🔧 استكشاف الأخطاء وإصلاحها (Troubleshooting)

#### ⚠️ المشكلة: عدم ظهور خلفية الشهادة عند التصدير إلى PDF.
* **الحل:** تأكد من أن الصور المستخدمة كقوالب مرفوعة مباشرة من جهازك أو مستضافة على سيرفر يدعم الـ CORS (سماح تبادل الموارد بين أصول مختلفة)، وإلا سيقوم المتصفح بحجبها لحمايتك.

#### ⚠️ المشكلة: ملف Excel لا يقرأ البيانات بالشكل الصحيح.
* **الحل:** تأكد من أن الصف الأول في ملف الـ Excel يحتوي على ترويسات الأعمدة (مثال: الاسم، اللقب، التقييم) دون صفوف فارغة بالأعلى، وأن الصيغة هي `.xlsx` أو `.xls`.

#### ⚠️ المشكلة: كلمة المرور لا تسمح بالدخول.
* **الحل:** يرجى التأكد من كتابة كلمة المرور بدقة وبنفس حالة الأحرف: `Aa@0555386421`.
